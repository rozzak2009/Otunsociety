# Assets Directory

Taruh seluruh gambar dan video milik Otunsociety Anda di sini:

1. **`assets/avatars/`**: Untuk menyimpan foto profil dari perangkat langsung (apabila real-time gagal / Anda tidak memakai integrasi internet aktif).
2. **`assets/gallery/`**: Untuk menyimpan karya mahakarya visual Anda (misal `edit1.mp4`, `color-grading.jpg`).

Lalu, **jangan lupa** Anda panggil lokasinya masuk ke dalam `data.js` misal: `src: "assets/gallery/karya1.jpg"`.
